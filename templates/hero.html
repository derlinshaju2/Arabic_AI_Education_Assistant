Frequently Asked Questions

Common questions about IntelliArabic

Learn how the image captioning and answer evaluation modules work.

What types of images can I upload?

You can upload common image formats such as JPG, JPEG, PNG and WebP, subject to the file restrictions configured by the application.

How is the Arabic caption generated?

The application first understands the image using a vision model and then translates the generated description into Arabic.

How does answer evaluation work?

The system compares the semantic meaning of the student's answer with the reference answer and converts the similarity into a score.

Does the student answer need to use exactly the same words?

No. The multilingual sentence model compares meaning, so answers can receive a strong score even when different wording is used correctly.
